### Disable days

You can disable some days by using the `isDaySelectable` prop:

<Playground>
  <p>
    Weekend days are disabled:
  </p>
  <Value>
    {({ value, set }) => (
      <Calendar
        value={value}
        onChange={set}
        isDaySelectable={date => {
          const weekday = date.getDay()
          return weekday !== 0 && weekday !== 6
        }}
      />
    )}
  </Value>
</Playground>

### Range

By using the prop `isDayWithinRange`, you can specify
a range area of days in your calendar.

This feature is use in the `InputDatePicker` component.

<Playground>
  <Calendar
    onChange={() => null}
    isDayWithinRange={(date, hoveredDate) => {
      return date.getDate() >= 10 && date.getDate() <= 20
    }}
  />
</Playground>

## Usage

```js
import { Calendar } from 'polvokit'

<Calendar
  value={new Date()} // selected date
  onChange={onChange} // change handler
/>
```
