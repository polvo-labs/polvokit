## Custom components

By default, it renders a react-router's Link component.
If you want to use `<a>` or `<button>` instead, you can use
the prop `as`:

<Playground>
  <Link as='a' href='#'>
    I'm an anchor
  </Link>
  <Link as='button'>
    I'm a button
  </Link>
</Playground>

## Usage

```js
import { Link } from 'polvokit'

// By default, it renders a react-router's Link
<Link to='/'>
  Home
</Link>

<Link as='a' href='https://example.com'>
  Example.com
</Link>

<Link as='button'>
  Click me
</Link>
```
