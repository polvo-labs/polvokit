# Checkbox

Just an alias for `<Option type='checkbox' {...props} />`.

<Playground>
  <VerticalSpacer>
    <Checkbox
      label='Check it'
    />
    <Checkbox

      defaultChecked
    />
  </VerticalSpacer>

## Usage

```js
import { Checkbox } from 'polvokit'

<Checkbox
  label='Checkbox label'
/>
```

## Props

It supports all props from `<input />`, plus:

<Props of={Option} />
