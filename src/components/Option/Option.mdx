# Option

A form component that can renders either a checkbox or a radio control.

  <VerticalSpacer>

      type='checkbox'
      label='Check it'
    />
    <Option
      type='checkbox'

</Playground>

<Playground>
  <VerticalSpacer>
    <Option

      label='Check it'
    />
    <Option
      name='radio'
      type='radio'
      label='Checked input'
      defaultChecked
    />
  </VerticalSpacer>
</Playground>

## Usage

```js
import { Option } from 'polvokit'

<Option
  type='checkbox'
  label='A checkbox label'
/>

<Option
  type='radio'
  label='A radio label'
/>
```

## Props

It supports all props from `<input />`, plus:

<Props of={Option} />
