## Usage

```js
import { Button } from 'polvokit'
```

## Custom components

You can use the prop `as` to render another component instead of a common button:

```js
// renders an anchor
<Button as='a' href='#'>
  I'm an anchor
</Button>
```

```js
// renders a react-router Link component
<Button as={Link} to='/'>
  I'm a Link
</Button>
```

However, the Button component provides two shortcuts to use these same components:

```js
<Button.Anchor href='#'>
  I'm an anchor
</Button.Anchor>
```

```js
<Button.Link to='/'>
  I'm an anchor
</Button.Link>
```

So you don't need to import { Link } from react-router to use Button as a Link.
