# ThemeDescriptor

Useful component for describing your theme config object.

## Usage

```js
import { ThemeDescriptor } from 'polvokit'

// Your theme should look like this:
const theme = {
  key: {
    subkey1: 'value'
    subkey2: 'value'
  },
  key2: {
    subkey1: 'value'
    subkey2: 'value'
  }
}

<ThemeDescriptor theme={theme} />
```
