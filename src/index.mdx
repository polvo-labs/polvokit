# Introduction

Polvokit is a React-based UI library.

## Installation

```
yarn add @polvo-labs/polvokit
```

You also have to install its peer dependencies:

```
yarn add styled-components styled-tools polished styled-media-query react-router-dom react-select
```

## Usage

Just import the components you want to use from the library:

```js
import { Button, Panel } from '@polvo-labs/polvokit'

export default function MyComponent () {
  return (
    <Panel title='Heading'>
      <Button>Click me</Button>
    </Panel>
  )
}
```
