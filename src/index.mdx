# Introduction

Polvokit is a React-based UI library.

## Installation

```
yarn add polvokit
```

You also have to install its peer dependencies:

```
yarn add styled-components styled-tools polished styled-media-query react-router-dom react-select date-fns msk
```

## Usage

Just import the components you want to use from the library:

```js
import { Button, Panel } from 'polvokit'

export default function MyComponent () {
  return (
    <Panel header='My Panel'>
      <Button>Click me</Button>
    </Panel>
  )
}
```
