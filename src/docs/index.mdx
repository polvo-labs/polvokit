# Introduction

Polvokit is a React-based UI library.

## Installation

```
yarn add polvokit
```

You also have to install its peer dependencies:

```
yarn add styled-components styled-tools polished react-router-dom react-select date-fns msk
```

## Usage

Just import the components you want to use from the library:

```js
import { Button, Card } from 'polvokit'

export default function MyComponent () {
  return (
    <Card title='My Card'>
      <Button>Click me</Button>
    </Card>
  )
}
```
